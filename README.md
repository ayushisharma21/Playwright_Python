# Parabank Playwright Automation Framework

This project automates 4 features of Parabank using Python + Pytest + Playwright.

## Features:
1. Login
2. Open New Account
3. Transfer Funds
4. Logout

## How to Run
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Install Playwright browsers:
   ```
   playwright install
   ```
3. Run all tests:
   ```
   pytest
   ```
